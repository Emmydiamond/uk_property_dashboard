# UK Property Price Analytics Dashboard with Price Prediction

## Overview

This project analyzes UK property price data through an interactive Streamlit dashboard. It allows users to explore property market trends and generate estimated property prices based on selected features such as district, year, property type, and tenure.

---

## 🎯 Project Objective

To build an interactive property analytics system that helps users understand UK housing trends and predict property prices using machine learning.

---

## 🚀 Features

* Interactive Streamlit dashboard for property analysis  
* Property price prediction based on user inputs  
* Real-time filtering of property data  
* Data visualization and trend exploration  
* Clean and user-friendly interface  

---

## ⚙️ How It Works

1. User selects property details (district, year, property type, tenure)  
2. Data is processed using Pandas  
3. Machine Learning model generates price prediction  
4. Streamlit displays results and insights interactively  

---

## 🤖 Machine Learning Model

* Algorithm: Regression Model (Scikit-learn)  
* Input Features: District, Property Type, Year, Tenure  
* Output: Estimated Property Price
